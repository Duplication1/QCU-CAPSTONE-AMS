Step 1: Firebase Setup

First, let's set up Firebase Realtime Database for real-time monitoring.

What you need to do:

  • Go to Firebase Console
  • Create a new project (or select existing)
  • Click "Realtime Database" ‚Üí "Create Database"
  • Choose location and start in test mode
  • Go to Project Settings ‚Üí Your apps ‚Üí Web app
  • Copy your Firebase configuration

Step 2: Configure Your Application

Now, update the Firebase configuration in your application files.

Files to update:

  • config/firebase_config.php - Backend configuration
  • pc-health/client/pc_health_agent.html - HTML Agent (line 165)
  • pc-health/client/start_agent.bat - PowerShell Agent (line 16)
define('FIREBASE_CONFIG', [
    'apiKey' => "YOUR_API_KEY",
    'authDomain' => "your-project.firebaseapp.com",
    'databaseURL' => "https://your-project-rtdb.firebaseio.com",
    'projectId' => "your-project",
    'storageBucket' => "your-project.appspot.com",
    'messagingSenderId' => "123456789",
    'appId' => "1:123456789:web:abc123"
]);

Step 3: Install Agents on PCs

Choose and install the monitoring agent on each laboratory computer.

Option A: HTML Agent (Easiest)

  • Open pc-health/client/pc_health_agent.html in browser
  • Fill in PC Unit ID, Terminal Name, and Room
  • Click "Start Monitoring"
  • Create shortcut in Startup folder to auto-start

Option B: PowerShell Agent (Most Reliable)

  • Edit start_agent.bat with PC details
  • Test by running the batch file
  • Place shortcut in Startup folder
  • Startup folder: C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp
